# Mi-Portafolio
Este proyecto es mi portafolio web desarrollado como parte de mi formación como estudiante de programación. Fue creado con el objetivo de aplicar y reforzar mis conocimientos en HTML, CSS y JavaScript, construyendo una página web completamente funcional y estructurada en archivos separados.

El sitio incluye navegación dinámica entre secciones, filtrado de proyectos y un formulario interactivo. Además, se trabajó el diseño visual para mantener una interfaz limpia, moderna y fácil de usar.

Este repositorio representa mi práctica en desarrollo web front-end y mi proceso de aprendizaje en la creación de aplicaciones web desde cero.
